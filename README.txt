Interaktív és reszponzív, szemantikus utazási emlék kezelő és tervező web alkalmazás.

Modul I.
Feltöltői fül:
A felhasználó feltöltheti már megtett utait és azokhoz tartozó képeit a web alkalmazásra, ami egy interaktív módon engedi, hogy emlékeit tárolja.

Modul II.
Interaktív színes térkép:
A már feltöltött utak alapján kiszínezett világtérkép. Az oldal tartalmaz több statisztikai nézetet is.

Modul III.
Utazás tervező kisokos:
A felhasználó érdekeltségei alapján kereshet megfelelő országokat.
